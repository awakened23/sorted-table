**SortedTable**

----------

Usage:

```html
<SortedTable
  data={someData}
/>
```
The data property needs to be an array of objects with the following object format:
id: String,
name: String,
family: String,
city: String,
score: Number

Here is an example:
```
[
  {id: 5, name: 'name1', family: 'family5', city: 'city5', score: 500},
  {id: 2, name: 'name1', family: 'family2', city: 'city5', score: 300}
]
```
